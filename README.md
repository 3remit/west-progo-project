
# West Progo Roleplay

Ini adalah lanjutan project dari basic gamemode [OpenMp_Gamemode](https://github.com/3remit/OpenMP_Gamemode) yang akan berfokus ke membangun gamemode RP.

Update akan di tulis pada [changelog](changelog.md)


## Authors

- [@3remit](https://github.com/3remit)


## Required Plugins

- [MySQL](https://github.com/pBlueG/SA-MP-MySQL) by [pBlueG](https://github.com/pBlueG) 
- [Samp Bcrypt](https://github.com/Sreyas-Sreelal/samp-bcrypt) by [Sreyas-Sreelal](https://github.com/Sreyas-Sreelal)
- [YSI Include](https://github.com/pawn-lang/YSI-Includes) by [Y-Less](https://github.com/pawn-lang)
- [Pawn.CMD](https://github.com/katursis/Pawn.CMD) by [katursis](https://github.com/katursis)
- [SSCANF](https://github.com/Y-Less/sscanf) by [Y-Less](https://github.com/Y-Less)
- [Streamer](https://github.com/samp-incognito/samp-streamer-plugin) by [samp-incognito](https://github.com/samp-incognito)
- [mSelection](https://github.com/vdgiapp/mSelection) by [vdgiapp](https://github.com/vdgiapp)
- [easyDialog](https://github.com/Awsomedude/easyDialog) by [Awsomedude](https://github.com/Awsomedude)
- [Discord connector](https://github.com/maddinat0r/samp-discord-connector) by [maddinat0r](https://github.com/maddinat0r)
- [Discord cmd](https://github.com/AkshayMohan/pawn-discord-cmd) by [AkshayMohan](https://github.com/AkshayMohan)
